## Como executar

1. Clone o repositório:
```bash
git clone https://github.com/FatecLP/api-admin-panel.git
```

2. Navegue até o diretório:
```bash
cd api-admin-panel
```

3. Instale as dependencias:
```bash
npm install
```

4. Transpile os scripts 
```bash
npx tsc
```

5. Execute a api
```bash
node dist/api/app.api.js
```
